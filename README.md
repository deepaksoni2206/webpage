# Webpage

This project showcases a dynamic webpage featuring a top menu, login page, and form. It was developed using HTML, CSS, Python (Django), and Bootstrap.

## Project Overview

The Webpage project demonstrates the implementation of a dynamic webpage using various web technologies. It includes a top menu that provides navigation options, 
a login page for user authentication, and a form for data input. The project utilizes HTML, CSS, Python (Django), and Bootstrap to create an interactive and visually
appealing webpage.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```shell
   $ git clone https://github.com/deepaksoni2206/webpage.git

2. Navigate to the project directory:

$ cd project1

3. Install the required dependencies. It is recommended to set up a virtual environment before installing the dependencies:

 $ python -m venv myenv  # Create a virtual environment (optional)
 $ source myenv/bin/activate  # Activate the virtual environment (optional)
 $ pip install -r requirements.txt  # Install project dependencies
 
4. Run the Django development server:
  
  $ python manage.py runserver

5. Open your web browser and visit http://localhost:8000 to access the webpage.

Usage: 

Once the project is set up and running, you can interact with the webpage in the following ways:

Navigate through the top menu to access different sections of the webpage.
Use the login page to authenticate yourself. 
Fill out the form to provide input and submit the data
